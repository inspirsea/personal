Api

Inputs

nodeItems: NodeItem<T>

callbacks: TreeCallbacks

options: TreeOptions

filter: string

Outputs

selectedItems: EventEmitter<T>

Methods

Add a new node to the tree
addNodeById(nodeItem: NodeItem<T>, id?: string): void

Delete a node from the tree
deleteById(id: string): void

Edit name of node.
editNameById(id: string, name: string): void

Edit item of node
editItemById(id: string, item: T): void

Expand all parents of a node
expandById(id: string): void

Collapse all parents of a node
collapseById(id: string): void

Trigger a select action of a node
selectById(id: string): void

Get the parent of a node
getParentById(id: string): NodeItem<T>

Expand all nodes
expandAll()

Collapse all nodes
collapseAll()

NodeItem<T>

Node identifier
id?: string

Name to be displayed
name: string

Item bound to the node, this is what will be returned on select
item?: T

If children is not set or 0, the node is a leaf
children?: NodeItem<T>[]

Determines if the node is selected from start. Default is false
selected?: boolean

Determines if the node is expanded from start. Only nodes that have children. Default is true
expanded?: boolean

TreeOptions

Which mode the tree should use, see TreeMode
mode: TreeMode

Determines if the tree should have checkboxes for displaying selected nodes
checkboxes: boolean

If true emits selectedItems on every select else only on init.
alwaysEmitSelected: boolean

TreeCallbacks

nameClick?: (item: NodeItem<T>) => void

select?: (item: NodeItem<T>) => void

unSelect?: (item: NodeItem<T>) => void

toggle?: (item: NodeItem<T>) => void

TreeMode

Dont allow item selection
NoSelect

Select one item at a time, if a second item is selected it will replace the current one
SingleSelect

Select multiple items
MultiSelect
