Custom Template

Customize the template by providing a ng-template in the content area. Currently customizable parts are:
nodeNameTemplate,
nodeCollapsibleTemplate

Customize nodeNameTemplate 
    
      // Identifier for customizing the nameTemplate.
      #nodeNameTemplate

      // Node will contain the nodeItem of the node.
      let-node="node"
      
      // Context will expose some methods, delete, toggleExpand
      let-context="context"

Example 
      
        {{customNameTemplateCode}}
{{node.name}} id: {{node.id}} delete
Customize nodeCollapsibleTemplate 
      
        // Identifier for customizing the collapsibleTemplate.
        #nodeCollapsibleTemplate
  
        // "expanded" determines if the node is expanded
        let-expanded="expanded"
Example 
    
      {{customCollapsibleCode}}
add remove
